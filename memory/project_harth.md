
1) تحسين التصميم والواجهة (UI/UX)

* تعديل ستايل الموقع بما يتناسب مع الهوية البصرية.
* ضبط الألوان، الأحجام، الخطوط، والمسافات بين العناصر.
* تحسين تجربة المستخدم وسهولة الاستخدام.



2) تحسين توافق الموقع مع جميع الشاشات (Responsive Design)

* مراجعة جميع الصفحات والتأكد من تناسقها مع:
    * الجوال.
    * الأجهزة اللوحية.
    * الكمبيوتر.
* إصلاح أي مشاكل في ظهور العناصر أو خروج الأزرار والصور عن الشاشة.



3) إكمال نظام تسجيل الدخول 

(فقط عن طريق جوجل مكتمل ) 

* تفعيل تسجيل الدخول باستخدام:
    * Google.
    * Apple.
* ربط الحسابات بشكل صحيح مع بيانات المستخدم.



4) إضافة نظام التحقق عن طريق الهاتف

* إضافة التحقق باستخدام رمز OTP عبر رقم الهاتف.
* استخدام التحقق عند إنشاء الحساب أو العمليات المهمة حسب الحاجة.



5) تعديل نظام التحقق من الحسابات

* مراجعة وتطوير نظام توثيق الحسابات.
* تحديد طريقة التحقق المطلوبة لكل نوع حساب.
* التأكد من صحة بيانات المستخدمين وتقليل الحسابات الوهمية.
* إضافة حالات للحساب مثل:
    * حساب غير موثق.
    * حساب موثق.
    * حساب يحتاج مراجعة.
* ربط التحقق بصلاحيات واستخدام خدمات الموقع.



6) إضافة نظام الإشعارات

* إنشاء نظام إشعارات للمستخدمين.
* يشمل:
    * تحديث حالة الطلبات.
    * قبول أو رفض الطلبات.
    * التنبيهات المهمة.
    * تحديثات الحساب.



7) مراجعة وتعديل شريط التنقل (Navbar)

* مراجعة ترتيب وتصميم شريط الأزرار.
* إجراء التعديلات المطلوبة إذا احتاج.
* (ليست أولوية حالياً).



8) تعديل نظام أنواع الحسابات

* إعادة تنظيم الحسابات لتكون:
    * حساب عادي (مستخدم).
    * حساب مدير النظام (Admin).
* إلغاء الفصل بين حساب المستهلك والمزارع.
* جعل الحساب العادي يمكنه استخدام الخدمات حسب احتياجه.



9) إعادة بناء نظام التوصيل

* مراجعة نظام التوصيل الحالي وإعادة تصميمه.
* تحديد:
    * مسؤول التوصيل.
    * طريقة حساب الرسوم.
    * متابعة حالة التوصيل.
    * ربط التوصيل مع الطلبات.



10) إضافة نظام الدفع الإلكتروني

* ربط الموقع مع بوابة دفع إلكترونية.
* دعم طرق الدفع المطلوبة.
* ربط الدفع مع الطلبات والعمليات المالية.
* إضافة حالات الدفع:
    * بانتظار الدفع.
    * تم الدفع.
    * فشل الدفع.
    * استرجاع المبلغ.
